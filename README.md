## Key Features

- **Local Processing**: Everything runs on your machine with Ollama AI models
- **ATS Compatibility Analysis**: Detailed resume compatibility scoring
- **Instant Match Score**: Quick analysis of resume vs job description
- **Keyword Optimization**: Identify and align with critical job keywords
- **Guided Improvements**: Clear, actionable suggestions

## Tech Stack

- **Frontend**: Next.js 15+, React 19, Tailwind CSS
- **Backend**: FastAPI (Python 3.12+)
- **AI**: Ollama (0.6.7+)
- **Database**: SQLite
